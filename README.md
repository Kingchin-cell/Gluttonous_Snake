# 🐍 贪吃蛇控制台游戏

![版本](https://img.shields.io/badge/版本-V1.1.1-green)

## 📖 项目介绍
纯C++开发Windows控制台经典贪吃蛇小游戏，代码结构清晰简洁，界面多彩美观，
搭载多线程按键监听、双缓冲地图渲染、独立难度最高分存档，
游玩体验流畅稳定，适合C++初学者学习参考。

## ✨ V1.1.1 更新内容
1. 护眼优化，取消全局频闪的刷新逻辑
2. 删除光标，避免光标乱飞导致的困扰

## ✨ V1.1.0 更新内容
1. 新增**三难度独立最高分系统**
2. 重构本地文件读写，分别存储简单/中等/困难最高分
3. 菜单界面配色优化，三色文字区分不同难度
4. 优化游戏结束分数判定逻辑，自动刷新对应难度记录
5. 界面文字排版美化，提升视觉体验

## 🎯 核心功能
- 🎨 多彩文字渲染：绿色蛇身、红色果实、蓝色难度标识
- ⚙️ 三种游戏难度：简单 / 中等 / 困难
- 💾 独立分数存档：各难度最高分分开保存，互不干扰
- 🚫 智能防掉头，遵循正统贪吃蛇规则
- 🧵 多线程按键监听，操控无延迟
- 🖼 地图缓存绘制，刷新干净不闪屏
- 🎯 食物智能生成，避开墙体与蛇身
- 💥 撞墙、撞自身双重游戏结束判定

## 🎮 操作按键
| 按键 | 移动方向 |
| ---- | -------- |
| W    | 向上移动 |
| S    | 向下移动 |
| A    | 向左移动 |
| D    | 向右移动 |

### 游戏规则
1. 吞噬果实增加蛇身长度，累计游戏得分
2. 触碰边界墙体、撞击自身身体直接结束游戏
3. 突破当前难度最高分，自动本地保存更新记录

## 🛠 运行环境
- 开发语言：C++
- 编译工具：Visual Studio 2022 / Dev-C++
- 运行系统：仅限 Windows 系统
- 依赖头文件：线程库、文件流、控制台按键、Windows延时API

## 🚀 快速启动
1. 克隆仓库至本地
2. 使用C++编译器打开源码
3. 直接编译运行进入游戏主菜单
4. 选择对应难度即可开始游玩
