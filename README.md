🐍 C++ 控制台贪吃蛇小游戏
 
项目简介
 
基于 C++ Windows控制台 开发轻量化经典贪吃蛇小游戏
代码结构简洁规范、界面彩色美化、运行流畅无卡顿
采用多线程按键监听、本地最高分存档、缓存渲染、智能碰撞判定，新手易读，适合C++入门学习
 
✨ 功能亮点
 
- 🎨 控制台彩色渲染：绿色蛇身、红色果实、白色默认字体
- ⚙️ 三档游戏难度：简单 / 中等 / 困难，速度自由切换
- 💾 最高分持久存档：自动读写本地文件，刷新记录永久保存
- 🚫 防反向掉头逻辑，符合正统贪吃蛇游玩规则
- 🎯 安全随机生成食物，不会刷在蛇身、墙体上
- 🧵 独立多线程监听按键，移动不卡顿、响应丝滑
- 🖼 双缓冲地图渲染，界面刷新干净无闪烁
- 💥 完整碰撞检测：撞墙、撞自身判定游戏结束
 
🎮 操作方式
 
按键 方向 
W 向上 
S 向下 
A 向左 
D 向右 
 
游戏规则
 
1. 吃掉苹果蛇身变长，累计本局得分
2. 撞击边界墙体、撞到自身身体，游戏结束
3. 本局分数高于历史最高，自动更新排行榜
 
🛠 运行环境
 
- 开发语言：C++
- 编译工具：Visual Studio 2022 / Dev-C++
- 适配系统：Windows 专属
- 依赖库：线程、文件流、控制台按键、Windows API
