贪吃蛇小游戏 · README
 
🎮 项目简介
 
基于 C++ 控制台 开发经典贪吃蛇休闲小游戏，采用面向对象思想+向量容器管理蛇身，实现方向操控、随机生成苹果、难度选择、积分本地存档、碰撞判定全套核心玩法，界面配色清新简洁，运行流畅无卡顿。
 
✨ 核心功能亮点
 
1. 三色主题渲染
自定义控制台文字配色，蛇身绿色、苹果红色、默认白色还原舒适视觉体验
2. 三档游戏难度
简单 / 中等 / 困难 自由选择，通过延时数值控制移动速度
3. 本地积分持久化
自动读取历史总分，游戏结束自动累加分数存入本地  score.txt  文件
4. 智能防逆向移动
禁止蛇直接掉头反向行走，贴合正版贪吃蛇游戏逻辑
5. 安全随机生果
苹果仅生成在空地，不会刷新在蛇身与墙体位置
6. 双线程按键监听
独立线程监听键盘操作，移动不卡顿、操控响应丝滑
7. 双层地图缓存渲染
原始地图+打印缓存分离，渲染效率高，界面刷新干净

🧩 核心技术要点
 
1. 坐标结构体封装
自定义  Pos  坐标类统一管理xy位置，代码整洁易维护
2. vector动态管理蛇身
无需固定数组，吃果自动增长蛇身长度，内存灵活可控
3. char二维数组地图
采用char类型存储地图状态，占用内存更小、读写速度更快
4. 转义字符控制台配色
极简  \033  颜色指令，直接搭配  cout  使用，无需复杂API
5. 文件流读写积分
 ifstream  读取历史分数， ofstream  写入最新总分，数据永久保存
6. 多线程分离业务
按键监听独立线程，主线程专注画面渲染与蛇移动逻辑
